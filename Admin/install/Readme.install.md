Install
============

After the base install, you should already have an up-to-date DB schema.


Updating
============

For an update of your installation, just apply the update files you didn't have already.  
The files need to be applied in ascending order.


Short description of the DB updates:
------------------------------------

| DB-Update      | Date       | Description                                                   |
|----------------|------------|---------------------------------------------------------------|
| `update-1.sql` | 2021-09-10 | Adds Faction selection for reporting and game history display |
